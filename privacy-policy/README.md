# Política de Privacidad - RouteNavEMT

Esta carpeta contiene la política de privacidad de la aplicación RouteNavEMT.

## Para publicar en GitHub Pages:

1. Crea un repositorio público en GitHub
2. Sube esta carpeta completa al repositorio
3. Ve a Settings → Pages
4. Selecciona la rama `main` como source
5. Tu URL será: `https://[tu-usuario].github.io/[nombre-repo]/`
